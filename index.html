🔥 خريطة Grad-CAM الحرارية

🧬 آلية عمل Grad-CAM خطوة بخطوة

1

تمرير أمامي (Forward Pass): تُمرَّر الصورة عبر الشبكة حتى الطبقة المستهدفة layer4[-1]

2

حساب التدرجات (Gradients): يحسب الموديل تدرج النتيجة المستهدفة بالنسبة لخرائط الميزات

3

الأوزان العالمية (Global Pooling): متوسط التدرجات يُعطي وزناً لكل خريطة ميزة

4

الدمج الرجحي: ضرب خرائط الميزات بأوزانها وجمعها ثم تطبيق ReLU

5

التكبير والدمج: تكبير الخريطة الحرارية لمقاس الصورة الأصلية ودمجها كتراكب شفاف

النتيجة: المناطق الحمراء في الخريطة تُمثل أكثر المناطق تأثيراً على قرار الموديل – مكان العيب بالضبط.

import cv2, numpy as np
from PIL import Image

def create_synthetic_bottle(is_defective=False):
    # إنشاء صورة سوداء (خلفية)
    img = np.zeros((224, 224, 3), dtype=np.uint8)
    
    # رسم شكل زجاجة (أسطوانة بيضاء)
    cv2.ellipse(img, (112, 112), (50, 100), 0, 0, 360,
              (255, 255, 255), -1)

    if is_defective:
        # إضافة كسر (خط أسود)
        cv2.line(img, (90, 80), (130, 100), (0,0,0), 5)
        # إضافة بقعة تلوث
        cv2.circle(img, (110, 150), 10, (0,0,0), -1)

    return Image.fromarray(img)

# توليد 100 صورة لكل صنف
for i in range(100):
    create_synthetic_bottle(False).save(f'dataset/good/bottle_{i}.png')
    create_synthetic_bottle(True).save(f'dataset/defect/defect_{i}.png')

import torch
import torch.nn as nn
from torchvision import models

# تحميل ResNet-34 مدرب مسبقاً على ImageNet
model = models.resnet34(weights='DEFAULT')

# تعديل الطبقة الأخيرة: 512 → 2 (سليم / معيب)
num_ftrs = model.fc.in_features
model.fc = nn.Linear(num_ftrs, 2)

# نقل للـ GPU إن وُجد
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = model.to(device)

print(f"✅ الموديل جاهز على: {device}")
